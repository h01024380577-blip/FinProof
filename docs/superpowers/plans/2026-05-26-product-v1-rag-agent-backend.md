# Product V1 RAG Agent Backend Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Persist Product V1 knowledge-base, RAG chunk, agent run, issue/evidence, chat, draft version, report, and case-library backend records while keeping the current mock-first Next.js backend stable.

**Architecture:** Keep route handlers thin and route all behavior through `ReviewService` and `ReviewStore`. Extend the mock store first for deterministic tests, then implement the same contracts in the Prisma store. Preserve current frontend compatibility by returning both canonical `items` pagination responses and legacy aliases where current UI code still depends on them.

**Tech Stack:** Next.js 16 App Router, TypeScript, Vitest, Prisma ORM 7, PostgreSQL, `@prisma/adapter-pg`, deterministic model/OCR providers by default.

---

## Scope Check

This plan covers one backend slice:

- Knowledge document metadata and approval.
- Evidence chunks.
- Agent run/finding persistence.
- Analysis result persistence into issues and evidence.
- Chat sessions/messages and mark-for-draft.
- Draft versions.
- Persisted markdown reports.
- Case library read API.
- Review list pagination/filter contract hardening.

It does not implement a full frontend admin UI, PDF/DOCX export, presigned multipart upload, or production pgvector operations.

  const normalizedPage = Number.isFinite(page) && page > 0 ? page : 1;
  const normalizedPageSize =
    Number.isFinite(pageSize) && pageSize > 0 ? Math.min(pageSize, 100) : 20;
  const start = (normalizedPage - 1) * normalizedPageSize;

  return {
    items: items.slice(start, start + normalizedPageSize),
    page: normalizedPage,
    pageSize: normalizedPageSize,
    total: items.length
  };
}

function reviewSummaryPage(
  items: ReviewSummary[],
  page?: number,
  pageSize?: number
): ReviewSummaryPage {
  const result = pageResult(items, page, pageSize);

  return {
    ...result,
    reviewCases: result.items
  };
}
