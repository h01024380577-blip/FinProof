내부 광고심의 체크리스트

요청부서: 디지털그로스팀
검토부서: 준법감시팀
상품명: 티리자정 수익플러스 계정
광고주: 미라공고
광고물: 온라인 포스터, SNS 이미지, 모바일 앱 배너
게시예정일: 2026-06-20

1. 상품명 및 광고주 표시
- 광고주명 표시: 예
- 상품명 또는 서비스명 표시: 부분 표시
- 실제 판매 주체와 광고주 관계 명확성: 검토 필요

2. 수익률 표시
- 포스터 문구: 무조건 연 5.5%
- 실제 조건: 목표수익률이며 확정수익률 아님
- 기본 판단: 부적정 가능성 높음
- 조치 필요: "목표 연 5.5% 수준", "운용성과에 따라 달라질 수 있음"으로 수정 필요

3. 원금 및 수익 보장 표현
- 포스터 문구: 전액보장, 무조건 수익, 손실 위험 제로
- 실제 조건: 원금손실 가능, 예금자보호 대상 아님
- 기본 판단: 부적정 가능성 매우 높음
- 조치 필요: 보장성 표현 삭제 및 투자위험 고지 확대 필요

4. 한도 및 지급 표현
- 포스터 문구: 한도 없음, 즉시 입금
- 실제 조건: 고객별 최대 5천만원, 지급 최대 3영업일 소요 가능
- 기본 판단: 실제 조건과 불일치
- 조치 필요: 가입한도와 지급 소요기간 명확화

5. 고지 문구
- 하단에 "수익률 확정 아님", "예금자보호법 보호되지 않음" 문구 있음
- 본문 보장성 표현이 지나치게 강해 하단 고지만으로 오인 가능성이 해소되기 어려움
- 모바일/SNS 축소 노출 시 하단 고지 가독성 부족 우려

6. 종합 요청 의견
- 현 시안은 조건부 승인보다 반려 또는 전면 수정 권고 가능성이 높음
- 보장성·무조건성·무위험성 표현 삭제 후 재심의 필요
- 투자성 상품 광고 기준, 금소법 광고규제, 표시광고법상 기만·과장 광고 여부 검토 필요
