This folder contains a realistic deposit/savings marketing review package for local FinProof intake testing.

Recommended intake metadata:

- 심의 요청 제목: 최고 연 5.0% JB 슈퍼씨드 적금 홍보물 심의
- 계열사: 광주은행
- 요청 부서: 디지털마케팅팀
- 상품군: 예금/적금
- 게시 예정일: 2026-06-20
- 게시 채널: 모바일 앱, 웹사이트
- 요청 메모: 최고금리 조건, 우대금리 산식, 예금자보호 문구, 중도해지 유의사항을 중점 검토해 주세요.
